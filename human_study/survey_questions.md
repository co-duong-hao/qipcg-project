# Blinded Level Rating Survey

Use this text to build a Google Form or Microsoft Forms survey. Keep generator
names hidden from participants.

## Opening Text

You will see a set of tile-based game levels generated by different algorithms.
The algorithms are hidden. Please rate each level based only on the grid shown.
There are no right or wrong answers; we are measuring perceived quality.

Estimated time: 15--25 minutes.

## Rating Scale

Use a 1--5 scale for each question:

- 1 = very poor
- 2 = poor
- 3 = acceptable / unclear
- 4 = good
- 5 = very good

## Questions Per Stimulus

For each stimulus ID, show the dataset name and level grid, then ask:

1. Playability: How playable or reachable does this level appear?
2. Style consistency: How well does this level fit the target game style?
3. Novelty: How distinct does this level feel compared with the other shown
   levels while still being coherent?
4. Overall quality: Overall, how good is this level?
5. Optional comment: What stood out about this level?

## Required Response Columns

If responses are exported to CSV manually, keep these columns:

```text
participant_id,stimulus_id,playability,style,novelty,overall,comment
```

If the self-contained HTML form is used, it exports exactly these columns.

The analysis script accepts these column names directly.
